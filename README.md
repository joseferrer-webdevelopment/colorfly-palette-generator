# ColorFly 🎨

Generador de paletas de colores aleatorias construido con HTML, CSS y JavaScript puro (vanilla).

🔗 **Demo en vivo:** https://joseferrer-webdevelopment.github.io/ProyectoM1_JoseFerrer/

Genera paletas de 6, 8 o 9 colores en formato HSL y HEX, con un solo clic puedes copiar cualquier código de color al portapapeles.

📄 **Documentación completa:** [README detallado](https://github.com/joseferrer-webdevelopment/ProyectoM1_JoseFerrer/tree/main/Documentacion)) — instrucciones de uso, decisiones técnicas, mejoras futuras y documentación del uso de IA en el desarrollo.
